# 🔐 GUI Password Generator (Python)

A simple and user-friendly **Password Generator Application** built using Python and Tkinter. This tool allows users to generate strong and random passwords based on selected criteria.

## 🚀 Features

* User-defined password length
* Option to include:

  * Uppercase letters (A–Z)
  * Lowercase letters (a–z)
  * Numbers (0–9)
  * Special characters (!@#...)
* Interactive GUI using Tkinter
* Instant password generation

## 🛠️ Tech Stack

* Python
* Tkinter (GUI)
* Random & String modules
